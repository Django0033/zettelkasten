# How to Write a Switch Statement in JavaScript

```js
let day;

// The getDay() method returns the weekday as a number between 0 and 6
// Expression
switch (new Date().getDay()) {
    case 0:
        // Code block for case 0
        day = "Sunday";
        break;

    case 1:
        // Code block for case 1
        day = "Monday";
        break;

    case 2:
        // Code block for case 2
        day = "Tuesday";
        break;

    case 3:
        // Code block for case 3
        day = "Wednesday";
        break;

    case 4:
        // Code block for case 4
        day = "Thursday";
        break;

    case 5:
        // Code block for case 5
        day = "Friday";
        break;

    case 6:
        // Code block for case 6
        day = "Saturday";
}
```
