# How to split a string in python?

```python
colors = 'red green blue yellow'
color_parts = colors.split() # Splits the colors string by spaces

print(colors)
print(color_parts)
```

```python
colors = 'red,green,blue,yellow'
color_parts = colors.split(',') # Splits the colors string by commas

print(colors)
print(color_parts)
```
