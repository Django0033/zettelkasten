# What does the pairs function do in lua?

It returns a stateles iterator that allows you to iterate over the key-value pairs of a table.

```lua
local arr = {'a','b','c'}

for i,v in pairs(arr) do
    print(i .. '/' .. v)
end

--[[ Prints...
1/a
2/b
3/c
]]
```
