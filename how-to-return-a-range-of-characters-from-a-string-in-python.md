# How to return a range of characters from a string in python?

```
b = 'Hello, World!'

print(b[2:5])
# Output -> llo

print(b[:5])
# Output -> Hello

print(b[2:])
# Output -> llo, World!

print(b[-5:-2])
# Output -> orl
```
